# VMinC
Création d une VM en C

1) Utilisation

Le programme se situe dans src

Pour compiler il faut faire:

```bash
make
```

Pour executer le programme:
```bash
./vm
```

L'option -h pourra être ajouté pour afficher l'aide

Le rapport se situe dans le dossier Rapport.
